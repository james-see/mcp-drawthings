mcp-drawthings

AI Image Generation for LLMs via Draw Things

npm v1.1.0 GitHub macOS Apple Silicon MCP

// QUICK INSTALL

npx -y mcp-drawthings

Click to copy

// TXT2IMG

Generate images from text prompts using Stable Diffusion models running locally on your Mac.

// IMG2IMG

Transform existing images with AI-powered modifications guided by text descriptions.

// LOCAL AI

All processing runs on your Mac using Apple Silicon. No cloud, no API costs, full privacy.

// MCP NATIVE

Built for Model Context Protocol. Works with Claude Desktop, Cursor, and any MCP client.

// AVAILABLE TOOLS

generate_image
Generate an image from a text prompt. Supports width, height, steps, cfg_scale, seed, and negative prompts.
transform_image
Transform an existing image using img2img. Control denoising strength for subtle or dramatic changes.
check_status
Verify Draw Things API server is running and accessible.
get_config
Get current Draw Things configuration including loaded model and settings.
